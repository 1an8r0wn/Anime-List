# Anime-List
展示新番的清單

https://gnehs.github.io/Anime-List/

## 如何使用
先載入資料
```
<script src="https://gnehs.github.io/Anime-List/anime2017.07.js"></script>
```
接下來就能開始使用了

## 範例
輸入:
```
Anime[0][0]
```
回傳:
```
DIVE!!跳水男孩
```
取得資料高度:
```
Anime.length
```
